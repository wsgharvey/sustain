A quick definition of the ELBO and manipulation into a form I like.

1 ELBO

The \(ELBO\) is defined as: \[ ELBO = \mathbb{E}_{q(x|y)} \left[ \log \frac{ p(x,y) }{ q(x|y) } \right], \] like a negative quasi-KL divergence between the approximate posterior and the joint.

We can show its nice properties as follows:

\begin{align*} \mathbb{E}_{q(x|y)} \left[ \log \frac{ p(x,y) }{ q(x|y) } \right] &= \mathbb{E}_{q(x|y)} \left[ \log \frac{ p(x|y) }{ q(x|y) } + \log p(y) \right] \\ &= \mathbb{E}_{q(x|y)} \left[ \log \frac{ p(x|y) }{ q(x|y) } \right] + \log p(y) \\ &= \log p(y) - D_{KL} \left( q(x|y) || p(x|y) \right) \end{align*}

So the ELBO is the exact log evidence minus the KL divergence between the approximate and true posterior.
