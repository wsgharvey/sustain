## To set up:

* Clone this repo

* Install jekyll with e.g. `https://websiteforstudents.com/how-to-install-jekyll-cms-on-ubuntu-16-04-18-04-18-10/`

* Run `bash build.sh`

## To set up org:

* sudo apt-get install bibtex2html

* copy over .org files and bibfile.bib 

* and more (e.g. configure .spacemacs)

* Build a single org file with ,eePf
